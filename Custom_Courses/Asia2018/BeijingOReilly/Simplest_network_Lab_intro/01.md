# 简单网络的设置

简单网络拥有值得注意的设置，因为它们将出现在所有神经网络中

* 学习速率
* 优化方法 (updater)
* 输出层激活函数
* 隐藏层激活函数
* 循环週期次数
* 误失函数

!SLIDE

# 从简单网络做推断


* 简单网络使用一个输入值和一个输出值，训练网络来预测正确的输出值
* 一般的神经网络会经过同样的过程与拥有更复杂的数据
* 输入数据可以是图像，文本，语音记录等等
* 输出数据可以是 T/F, 连续值, 标签
